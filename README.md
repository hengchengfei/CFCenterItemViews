## Requirements

- iOS 8.0+
- Xcode8+
- Swift 3

## Installation

CFCenterItemViews is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "CFCenterItemViews"
```
